golang中的一些常用的內建变量类型

  • bool, string
  • (u)int, (u)int8, (u)int16, (u)int32, (u)int64, uintptr
  • byte, rune
  • float32, float64, complex64, complex128 golang中没有char类型，只有rune类型

首先在golang中是没有隐式类型转换的，在Python之类的语言当中解释器会将类型自动转换，而golang中是需要手动强制进行转换的。

func triangle(){
    var a, b int = 3, 4
    var c int
    c = int(math.Sqrt(float64(a * a + b * b))
    fmt.Println(c)
}

因为变量c的类型是int，所以在计算其结果(float64)之后要将它放入c，就需要将结果强行转换成int，golang编译器是不会自动将它转换成int的。

条件判断

  • IF语句
func condition_judgment(){
    const filename = "abc"
    if contents, err = ioutils.ReadFile(filename); err == nil{
        fmt.Println(contents)
    }else {
        fmt.Println("canot open file: ", err)

首先在golang的if语句中，是可以进行赋值操作的，但赋值的作用于仅限于if作用域。

  • Switch语句
func grade(score int) string {
	g := ""
	switch  {
	case score < 0 || score > 100:
		panic(fmt.Sprintf("Wrong score: %d", score))
	case score < 60:
		g = "F"
	case score < 80:
		g = "C"
	case score < 90:
		g = "B"
	case score <=100:
		g = "A"
	}
	return g
}

在golang中的switch和其他语言(Java, C)的switch作用差不多，但是golang中的switch没有break语句，break会自动执行，如果需要去除break，则需要使用fallthrough。

循环(loop)

  • for语句
// 手动读取文件
func printFileContent(filename string) {
	file, err := os.Open(filename)
	if err != nil {
		panic(err)
	}
	scanner := bufio.NewScanner(file)
	for scanner.Scan(){
		fmt.Println(scanner.Text())
	}
}

// 计算整数的二进制码
func convertToBin(n int) string {
    result := ""
    for ; n > 0; n /= 2{
        lsb := n % 2
        result = strconv.Itoa(lsb) + result //将lsb转换成字符串
    }
    return result
}

在golang中是没有while语句的，因为while语句和for极其相似，所以被golang的设计者给合并了，看起来也很有道理。如果要使用while语句的功能，可以使用如下写法，将条件语句和结束语句省略。

// 永真循环， 可当while使用
func forever() {
    for {
        fmt.Println("abcd")
    }
}

函数

  • 示例1
func eval(a, b int, op string) (int, error){
	switch op {
	case "+":
		return a + b, nil
	case "-":
		return a - b, nil
	case "*":
		return a * b, nil
	case "/":
		q, _ := div(a, b)
		return q, nil
	default:
		return 0, fmt.Errorf("unsupported operation: %s", op)
	}
}

func div(a, b int) (int, int){
	return a / b, a % b

函数中可以传入一个error类型用于错误处理

  • 示例2
func pow(a, b int) int {
	return int(math.Pow(float64(a), float64(b)))
}

func apply(op func(int, int) int, a, b int) int{
	p := reflect.ValueOf(op).Pointer()
	opName := runtime.FuncForPC(p).Name()
	fmt.Printf("Calling function %s with args "+ "(%d, %d)\n", opName, a, b)
	return op(a, b)
}

首先我重写了示例1的代码，将它变得看起来更优雅(唬人)，因为golang是一门函数式编程语言，所以我们可以在函数里面传入函数，函数名字写在前面op func(int, int) int这样，最后返回这个传入的函数结果，类型为int。

调用该函数

fmt.Println(apply(pow, 3, 4))

另一种写法，可以在调用的时候直接写入一个匿名函数(golang中没有函数重载，lambda之类的操作)。

fmt.Println(
    apply(func(a int, b int) int{
        return int(math.Pow(float64(a), float64(b)))
    }, 3, 4))

传入多个参数

func sum(numbers ... int) int {
	s := 0
	for i := range numbers {
		s += numbers[i]
	}
	return s
}

fmt.Println(sum(1, 2, 3, 4, 5))

output:
    15

在golang创建函数的时候参数的建立有一点我觉得是非常不好的，如下

func demoCode(str string, x y z int, c bool){
    ...

一旦参数多了就非常扰乱视线！
如果不嫌麻烦可以一个参数后面跟着一个类型。

指针

在golang中的指针使用是非常简单的，并且不能对它做运算操作。

  • 经典示例
func swap(a, b int) {
    *a, *b = *b, *a
}
a, b = 3, 4
swap(&a, &b)

当然，不推荐这种写法，只是做一个示例。

func swap(a, b int) (int, int) {
    return b, a
}

写成这样或许更加优美一些
