关于向量代数的一些笔记

这是一篇没有技术含量的文章...

概述

在平面解析几何当中, 通过坐标法将平面上的点与一对有次序的数对应起来, 把平面上的图形和方程对应起来, 从而可以用代数方法来研究几何问题。空间解析几何也是按照类似的方法建立的。

平面解析几何的只是对于学习一元函数微积分是不可以缺失的学习, 空间解析几何的知识对于学习多元函数微积分也是十分必要的。

行向量与列向量

行向量: \((x, y)\)
列向量: \(\dbinom{x}{y}\)

目前两者没啥分别, 在用到矩阵的时候会区别开来, 而通常提到的向量一般都是列向量

由于很多教材书籍时横向印刷的, 表示为\((x, y)^T\), 和上面一个意思

与之向量相反的是标量

向量之间的操作

加法

\((x, y)^T\) + \((a, b)^T\) = \((x + a, y + b)^T\)

乘法

\(k\times(x, y)^T\) = \((kx, ky)^T\)

数量乘法

\(k \cdot\left\{\begin{matrix} v1 \\ v2 \\ ... \\ v_n \\ \end{matrix}\right\}\) = \(\left\{\begin{matrix} k\cdot v1 \\ k\cdot v2 \\ ... \\ k\cdot v_n \\ \end{matrix}\right\}\)

n维向量也是如此

1、向量及其线性运算

既有大小，又有方向的量叫做向量（或矢量）

在数学上，常用有向线段来表示向量。有向线段的长度表示向量的大小，有向线段的方向表示向量的方向，注意向量的记法（坚持使用箭头记法）

在数学上，我们只研究与起点无关的向量，并称这种向量为自由向量。如果遇到与起点有关的向量时，可在一般原则下作特别处理

如果两个向量\(\overrightarrow{a}\)和\(\overrightarrow{b}\) 的大小相等，且方向相同，则两个向量相等，记作\(\overrightarrow{a}\) = \(\overrightarrow{b}\)

向量的大小叫做向量的模

模等于1的向量叫做单位向量(\(|\overrightarrow{a}|\)=1)

模等于0的向量叫做零向量，记作\(\overrightarrow{0}\) ，零向量的起点和终点重合，它的方向可以看做是任意的（方向任意）

两向量之间的夹角在0到π之间（可相等），记法略

如果两向量之间夹角为\(0\)或\(\pi\)，就称两向量平行，记作\(\overrightarrow{a} // \overrightarrow{b}\) 。如果两向量之间的夹角为\(\pi2\)那么称两向量垂直，记作\(\overrightarrow{a} \bot \overrightarrow{b}\) 可以认为零向量与任何向量都平行，也可以认为零向量与任何向量都垂直

当两个平行向量的起点放在同一点时，它们的终点和公共起点应在一条直线上，因此，两向量平行，又称两向量共线

设有\(k(k\ge3)\)个向量，当把它们的起点放在同一点时，如果k个终点和公共起点在一个平面上，就称这\(k\)个向量共面

向量加法的三角形法则或平行四边形法则

向量的加法符合交换律和结合律**

设\(\overrightarrow{a}\) 为一向量，与\(\overrightarrow{a}\) 的模相同而方向相反的向量叫做\(\overrightarrow{a}\) 的负向量，记作\(-\overrightarrow{a}\) 注意向量加减法的几何理解

由三角形两边之和大于第三边有

\[|\overrightarrow{a} + \overrightarrow{b}| \leq|\overrightarrow{a}| + |\overrightarrow{b}|\]

\[|\overrightarrow{a} - \overrightarrow{b}| \leq|\overrightarrow{a}| + |\overrightarrow{b}|\]

其中等号在\(\overrightarrow{a}\) 与\(\overrightarrow{b}\) 同向或反向时成立 向量的数乘符合结合律和分配律

向量数乘的结果仍然是一个向量，数乘\(\lambda\overrightarrow{a}\) 的模是
\(|\lambda\overrightarrow{a}| = |\lambda| |\overrightarrow{a}|\)

向量相加及数乘向量统称为向量的线性运算

通常用\(\vec e_{\vec a}\) 表示与非零向量\(\vec a\) 同方向的单位向量

一个非零向量除以它的模的结果是一个与原向量同方向的单位向量，即\(\vec a |\vec a| = \vec e_{\vec a}\)
定理1：设向量\(\vec a \ne \vec 0\) ，则向量\(\vec b\) 平行于\(\vec a\) 的充分必要条件是：存在唯一的实数λ，使\(\vec b = \lambda \vec a\) ，\(\lambda\)可为\(0\)
给定一个点及一个单位向量就确定了一条数轴，数轴上的点与实数之间有一一对应的关系，\(\vec {OP}= \vec {xi}\)

空间直角坐标系

空间直角坐标系也称为\(Oxyz\)坐标系，其符合右手规则(右手握拳做出‘你真棒’的手势)

注意空间直角坐标系卦限的分布(顺序为逆时针)

点\(M\)、向量\(\vec \gamma\) 与三个有序数\(x\)、\(y\)、\(z\)之间有一一对应的关系

点的坐标，向量的坐标

向量\(\vec \gamma = \vec {OM}\)称为点M关于原点O的向径。一个点和该点的向径（向径一般都是相对于原点的）有相同的坐标。但是，在几何中点和向量是不一样的概念，一定要注意区分

坐标面上和坐标轴上的点，其坐标各有一定的特征
